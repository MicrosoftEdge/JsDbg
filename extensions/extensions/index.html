Looking to edit an extension?
  1. Copy the extension's directory locally.
  2. If the extension is loaded, unload it below.
  3. Load your local copy by entering the path above.
  4. Contact the extension author to contribute your changes.
If you want to edit multiple built-in extensions, it may be easier to work from a local copy of the entire extension root:
  1. Clone the repository from the JsDbg project, or copy the default extension root (below) to your local machine.
  2. Change the extension root below to your local copy.
  3. Files will now be served from your private copy, where you can edit them freely.
  4. Once you are satisfied with your changes, commit them to a new branch and open a pull request on VSTS!
